实验一、思科路由器的开机登陆密码设置
Would you like to enter the initial configuration dialog? [yes/no]: n
//如果输入yes，按CTRL+C退出

Router>
//用户模式，特权等级为1，这个模式只能简单查看设备

Router>sh pri
Current privilege level is 1

Router>en
//全称是enable，作用是将设备从用户模式提升到特权模式
Router#
//特权模式，特权等级为15，是最高权限

Router#sh clock
*0:12:11.32 UTC Mon Mar 1 1993
//查看设备当前时间和日期
Router#clock set 11:43:00 21 Feb 2019
//修改设备当前时间和日期
Router#sh clock
11:43:3.816 UTC Thu Feb 21 2019

Router#conf t
//将设备从特权模式改变成全局配置模式
Router(config)#
//全局配置模式

Router(config)#ho R1
//全称是hostname R1，修改设备主机名为R1

R1(config)#no ip domain loo
R1(config)#line c 0
R1(config-line)#no exec-t
R1(config-line)#logg s
R1(config-line)#exi

开机登陆密码：
1、明文密码：enable password XXXXXX
2、密文密码：enable secret XXXXXX
注意：一般情况下两种密码不会同时设置，如果同时设置的话，明文密码无效，密文密码有效。

R1(config)#ena pass cisco
R1(config)#ena sec 123456

R1(config)#exi
//退出，exit是一级一级退出，end是从任何一级直接退到特权模式
R1#wr
//全称是write，保存配置

测试密码：
R1#exi
R1>en
Password: 输入密码cisco，无法进入设备
Password: 输入密码123456，可以进入设备
R1#

R1#sh run
!
enable secret 5 $1$mERr$H7PDxl7VYMqaD3id4jJVK/
enable password cisco
!
//按回车是一行一行显示，空格是一大段一大段显示，也可以按CTRL+C中断显示

实验三、远程登陆方式之Telnet

1、线缆
·网线（电口）
  直连线：568B ---- 568B
  交叉线：568A ---- 568B

  568A：绿白 绿 橙白 蓝 蓝白 橙 棕白 棕（1/3、2/6）
  568B：橙白 橙 绿白 蓝 蓝白 绿 棕白 棕

  同种设备之间使用交叉线；不同种设置之间使用直连线；例外电脑或者服务器连接路由器使用交叉线

·光纤（光口）
  多模光纤：两公里之内使用，发光源是二级发光管
  单模光纤：最大100公里使用，发光源是激光

2、IP地址
192.168.10.1/24和192.168.10.2/24就能通信？

二进制（0、1）和十进制（0 - 255）的转换：
128 64 32 16 8 4 2 1

11000000 10101000 00001010 00000001（192.168.10.1）
11111111 11111111 11111111 00000000（255.255.255.0）
-----------------------------------
11000000 10101000 00001010 00000000（192.168.10.0）

1和1相与是1；1和0相与是0；0和0相与是0

11000000 10101000 00001010 00000010（192.168.10.2）
11111111 11111111 11111111 00000000（255.255.255.0）
-----------------------------------
11000000 10101000 00001010 00000000（192.168.10.0）

通过和掩码相与，两个地址网段相同，则可以相互通信

一、基本配置
Router>en
Router#conf t
Router(config)#ho R1

二、优化配置
R1(config)#no ip domain loo
R1(config)#line c 0
R1(config-line)#no exec-t
R1(config-line)#logg s
R1(config-line)#exi

三、接口配置
R1(config)#int g0/0
//全称是interface g0/0
R1(config-if)#ip add 192.168.10.1 255.255.255.0
//全称是ip address 192.168.10.1 255.255.255.0
R1(config-if)#no sh
//路由器接口默认关闭，需要手动开启，命令是no shutdown
R1(config-if)#exi

·10Mbps接口：E0/0（全称是Ethernet）
·100Mbps接口：F0/0（全称是FastEthernet）
·1000Mbps接口：G0/0（GigabitEthernet）,也叫1Gbps
·10000Mbps接口：Ten-G0/0，也叫10Gbps

·b：指bit，比特，速率一般使用比特
·B：指Byte，字节，存储一般使用字节（1B=8b）

带宽：100Mbps，换算成存储单位就是12.5MBps

四、测试直连
R1(config)#exi
R1#ping 192.168.10.2
//第一次Ping一定是.!!!!，以后Ping都是!!!!!

五、配置远程访问
1、定义设备的开机登陆密码
R1#conf t
R1(config)#ena sec cisco
2、开启设备的远程访问功能
R1(config)#line vty 0 4
//开启远程访问，默认是Telnet，0 4代表开启设备5个线程，可以同时运行5个用户远程访问
R1(config-line)#pass 123456
//定义远程登陆密码是123456
R1(config-line)#login
//启用远程登陆认证，认证本地密码
R1(config-line)#transport input telnet 
//具体制定远程登陆的方式是Telnet
R1(config-line)#exi

实验四、远程访问之SSHv2
IPv4地址的分类：
A类：1-126开头，默认掩码/8
B类：128-191开头，默认掩码/16
C类：192-223开头，默认掩码/24
D类：224-239开头，组播地址
E类：240-255开头，科研保留

127：一般作为环回地址使用，特别是127.0.0.1

VLSM：可变长子网掩码技术
CIDR：无类域间路由

1、基本配置
Router>en
Router#conf t
Router(config)#ho R1
2、优化配置
R1(config)#no ip domain loo
R1(config)#line c 0
R1(config-line)#no exec-t
R1(config-line)#logg s
R1(config-line)#exi
3、接口配置
R1(config)#int g0/0
R1(config-if)#ip add 192.168.10.1 255.255.255.0
R1(config-if)#no sh
R1(config-if)#exi
//进入接口以后，可以先开启接口，再配置IP地址，这个是个人习惯
4、测试直连
R1(config)#exi
R1#ping 192.168.10.2
//现象是.!!!!，第一个.代表ARP地址解析，将IP地址解析成MAC地址
5、开启远程访问
·配置设备的开机登陆密码
R1#conf t
R1(config)#ena sec cisco
·配置设备的主机名（Hostname）和域名（Domain-name）
R1(config)#ip domain-name cisco.com
//配置主机名和域名的原因是密钥的名字是主机名和域名组成
·生成RSA不对称加密密钥，密钥长度是1024Bit
R1(config)#crypto key generate rsa general-keys modulus 1024

注意：The name for the keys will be: R1.cisco.com

注意：*3月 1 0:28:10.898: %SSH-5-ENABLED: SSH 1.99 has been enabled
//1024就是开启SSHv2
·优化SSH参数
R1(config)#ip ssh authentication-retries 3
R1(config)#ip ssh time-out 10
R1(config)#ip ssh version 2
·配置用户名和密码
R1(config)#username abc pass 123456
·配置远程访问
R1(config)#line vty 0 4
R1(config-line)#login local
//login指验证密码；login local指验证用户名和密码
R1(config-line)#tr in ssh
R1(config-line)#exi
·测试

==============================================================

1、路由器（Router）
2、交换机（Switch）
·二层交换机：工作在OSI参考模型的数据链路层
·三层交换机：默认工作在OSI参考模型的数据链路层，可以在全局通过命令ip routing和在接口通过命令no switchport开启网络层的功能
·多层交换机：同时开启路由进程和交换进程，既可以当成路由器使用，也可以当成交换机使用

·二层交换机：C2960系类
·三层交换机：C3560系类和C3650系类

OSI参考模型：
·应用层
·表达层
·会话层：操作系统工作在这一层
·传输层（TCP/UDP，单位是数据段Segment）
·网络层（IP，单位是数据包Packet）
·数据链路层（将0和1组成数据帧Frame，单位是Frame）
·物理层（传输0和1，单位是比特Bit）

TCP/IP协议模型（四层模型和五层模型）
应用层：会话层、表达层、应用层
主机到主机层：传输层
Internet层：网络层
接入层：物理层和数据链路层

五层是接入层分为物理层和数据链路层

实验五、VLAN基本配置
LAN：局域网
MAN：城域网
WAN：广域网
SAN：存储网
VLAN：虚拟局域网
WLAN：无线局域网

WIFI：组织，机构

一、基本配置和优化配置
Switch>en
Switch#conf t
Switch(config)#ho SW1
SW1(config)#no ip domain loo
SW1(config)#line c 0
SW1(config-line)#no exec-t
SW1(config-line)#logg s
SW1(config-line)#exi

二、创建VLAN（范围是0-4095，一共有4096个VLAN）
·VLAN 0：系统默认，不能创建
·VLAN 1：系统默认，不能创建
·VLAN 1002 - 1005：系统默认，不能创建
·VLAN 4095：系统默认，不能创建
·常用的VLAN范围是：2 - 1001

2^12 = 4096

SW1(config)#vlan 10
SW1(config-vlan)#name TECH
SW1(config-vlan)#vlan 20
SW1(config-vlan)#name SALE
SW1(config-vlan)#vlan 30
SW1(config-vlan)#name HR
SW1(config-vlan)#exi

三、接口加入VLAN
SW1(config)#int f0/1
SW1(config-if)#sw mo ac
SW1(config-if)#sw ac vlan 10

SW1(config-if)#sw mo ac
