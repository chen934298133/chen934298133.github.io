实验一：思科路由器开机密码设置
实验二：思科交换机开机密码设置

模拟器：
·初级：Cisco PT 7.0
·中级：GNS3、EVE-NG
·高级：IOU（IOS Over UNIX）或者IOL（IOS Over LINUX）

·华为：eNSP
·华三：HCL

·收费：Cisco VIRL

====================================================

笔记本电脑
配置线：也叫Console线
转换线：USB转RS232或者USB转COM口的线
软件：SecureCRT（收费、主流）、puTTY（免费）

====================================================

路由器：
ISR-1：1800、2800、3800系列，主要2811
ISR-2：1900、2900、3900系列，主要2911
ISR-3：4000系列，主要4321

交换机：
二层交换机：2960系列
三层交换机：3560系列、3650系列

1、引擎：CPU
2、主板
3、内存：RAM（随机存储器）
4、闪存：FLASH，相当于硬盘
5、ROM：只读存储器，相当于电脑的BIOS
6、NVRAM：非易失性随机存储器，主要用来存储配置
7、端口：光口、电口、串口

====================================================

系统：
·思科：IOS（互联网操作系统）
·詹博：JUNOS
·华为/华三：VRP（通用路由平台）
·锐捷：ROS

====================================================

实验一、思科路由器的开机登陆密码设置
Would you like to enter the initial configuration dialog? [yes/no]: n
//如果输入yes，按CTRL+C退出

Router>
//用户模式，特权等级为1，这个模式只能简单查看设备

Router>sh pri
Current privilege level is 1

Router>en
//全称是enable，作用是将设备从用户模式提升到特权模式
Router#
//特权模式，特权等级为15，是最高权限

Router#sh clock
*0:12:11.32 UTC Mon Mar 1 1993
//查看设备当前时间和日期
Router#clock set 11:43:00 21 Feb 2019
//修改设备当前时间和日期
Router#sh clock
11:43:3.816 UTC Thu Feb 21 2019

Router#conf t
//将设备从特权模式改变成全局配置模式
Router(config)#
//全局配置模式

Router(config)#ho R1
//全称是hostname R1，修改设备主机名为R1

R1(config)#no ip domain loo
R1(config)#line c 0
R1(config-line)#no exec-t
R1(config-line)#logg s
R1(config-line)#exi

开机登陆密码：
1、明文密码：enable password XXXXXX
2、密文密码：enable secret XXXXXX
注意：一般情况下两种密码不会同时设置，如果同时设置的话，明文密码无效，密文密码有效。

R1(config)#ena pass cisco
R1(config)#ena sec 123456

R1(config)#exi
//退出，exit是一级一级退出，end是从任何一级直接退到特权模式
R1#wr
//全称是write，保存配置

测试密码：
R1#exi
R1>en
Password: 输入密码cisco，无法进入设备
Password: 输入密码123456，可以进入设备
R1#

R1#sh run
!
enable secret 5 $1$mERr$H7PDxl7VYMqaD3id4jJVK/
enable password cisco
!
//按回车是一行一行显示，空格是一大段一大段显示，也可以按CTRL+C中断显示
